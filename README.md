# GitHub User Finder

**Автор:** Николай Сорокин

## Описание

GUI-приложение для поиска пользователей GitHub с возможностью сохранения избранных аккаунтов. Программа использует официальное GitHub API для поиска пользователей по логину и позволяет добавлять понравившихся пользователей в избранное с локальным сохранением в JSON-файл.

### Основные возможности

- 🔍 Поиск пользователей GitHub по имени
- 📋 Отображение результатов в удобной таблице (логин, ID, рейтинг)
- ⭐ Добавление пользователей в избранное
- 💾 Автоматическое сохранение избранных пользователей в JSON
- ❌ Удаление пользователей из избранного
- ✅ Валидация поля поиска (не может быть пустым)

## Технологии

- **Python 3.x**
- **tkinter** - для графического интерфейса
- **requests** - для работы с GitHub API
- **JSON** - для хранения избранного
- **Git** - для контроля версий
